HTML

  • HTML: Hypertext Markup Language is used to create the structure of a webpage.
  • HTML elements is a type of content on a webpage. They include:
    • Text elements
    • Image elements
    • Header elements
    • List elements
  • Other types of elements are also used to divide the content on the page and describe the content to browsers and other developers. These include:
    • Meta-related elements
    • Content division elements
    • Semantic elements
  • The head element contains information about the webpage.
  • The body element represents the visibile contentshown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The most common way to style a webpage is by using an external CSS style sheet
    • When using an external style sheet, we link the file using a <link> element placed in the HTML's <head>
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • Sometimes, we want to apply a rule to all the elements visible on the page. To do that, we use a special wild card symbol, or *

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds all modifications in the current working branch to the staging area.
  • git commit -m "[descriptive message]": commit staged content as a new commit snapshot. Once commited the current branch will be updated with modifications.
  • git pull origin main: syncs with the remote base branch. We do this to insure working on the most up-to-date code
  • git push origin [branch]: transmit local branch commits to the remote repository branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
    • A function's name is followed by parentheses.
    • Functions do not automatically execute when the JavaScript file runs.
    • A function must be called by name in order to be executed.
